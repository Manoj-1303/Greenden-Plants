# 🌿 Greenden – Plant & Pots Website

**Greenden** is a clean and responsive static website built using **HTML** and **Tailwind CSS**. It is designed for showcasing a collection of plants and pots with a modern, minimal user interface.

## 🚀 Features

* Responsive design for mobile, tablet, and desktop
* Clean and modern UI using Tailwind CSS
* Easy navigation across multiple pages
* Lightweight and fast-loading static site

## 📄 Pages Included

* **Home Page**
  A visually appealing landing page introducing Greenden and highlighting featured plants and offers.

* **Products Page**
  Displays a variety of plants and pots with organized layouts for easy browsing.

* **Contact Page**
  Simple contact section for users to connect or send inquiries.

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS

## 📌 Purpose

This project is a beginner-friendly static website focused on practicing frontend development, layout structuring, and responsive design using Tailwind CSS.

A simple and elegant project to showcase plant products with a user-friendly experience.
